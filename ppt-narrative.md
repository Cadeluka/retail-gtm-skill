# PPT叙事结构参考

## 核心叙事框架

```
诊断现状 → 识别症结 → 开出药方
（市场表现总览）→（核心问题洞察）→（战略方向思考）
```

这是一种"侦探式"叙事：先呈现数字事实，再层层剥开"谁造成了问题"，最后给出有针对性的行动建议。

---

## 标准幻灯片清单（18-22张）

### 封面（1张）
```
主标题：[结构重塑与增量突围] 或类似
副标题：20XX年Q[X]零售业务回顾与增长策略
汇报人：[姓名] | [职位]
```

### 目录（1张）
```
1. 市场表现总览
2. 核心问题洞察  
3. 战略方向思考
```

---

### Part 1：市场表现总览（3-4张）

#### Slide 3：章节标题页
```
大字：市场表现总览
```

#### Slide 4：整体业绩回顾
**标题**：整体业绩回顾：[一句话定性总结，如"总量承压，渠道sell out有所改善"]

**布局**：
- 左上：两个KPI大数字
  - 全年Sell-in：X.XX亿 / YoY±XX%
  - 全年Sell-out：X.XX亿 / YoY±XX%
- 主体：月度进销趋势折线图（Sell-in + Sell-out双线）
- 右侧/下方：产品指标表格（各产品YTD Sell-in/Sell-out）
- 底部：核心洞察文字框

**核心洞察文字**（必须包含）：
- Sell-in vs Sell-out的相对关系（谁降幅更大？）
- 月度趋势的关键特征（如哪个月出现拐点）
- Q1+Q2 vs Q3+Q4的节奏对比

#### Slide 5：区域表现
**标题**：区域表现：两极分化，[A省]的"崩盘"与[B省]的"坚守"形成鲜明对比

**布局**：
- 左侧：省份Sell-in增长贡献瀑布图/柱状图
- 右侧：产品维度省份增长贡献图
- 核心洞察：点出"市场下滑不是共性问题"

---

### Part 2：核心问题洞察（5-7张）

#### 章节标题页
```
大字：核心问题洞察
```

#### Slide 7：产品季度矩阵（时间维度 × 产品维度）
**标题**：核心洞察（一）：核心产品早期Sell-out表现需重点关注

**布局**：
- 上半部分：分季度×分产品同比下滑表格（单位：万，标注增长率）
- 下半部分：时间轴洞察（Q1+Q2贡献了全年X%的下滑）
- 产品维度洞察（F产品贡献X%的下滑，R产品逆势增长）

**表格格式**：
```
产品  Q1            Q2            Q3           Q4          全年
F    -X,XXX(-XX%)  -X,XXX(-XX%)  ...          ...         -X,XXX(-XX%)
C    ...
R    +XXX(+XX%)    ...（高亮绿色）
汇总  ...
```

#### Slide 8：产品进销配比（产品健康度）
**标题**：产品洞察拆解：产品Sell-in和out情况分析

**布局**：
- 散点图或双柱图：各产品的SI vs SO对比
- 重点标注：R产品SO > SI（渠道健康信号）
- 重点标注：F/C产品主要下滑来源的TOP连锁客户

**核心洞察**：
- "R产品在核心DTP渠道存在进一步发展潜力"
- "F产品下滑主要来自[客户A]（XXXX万）和[客户B]（XXXX万）"

#### Slide 9：产品潜力分析（聚焦新增长）
**标题**：产品洞察拆解：R产品贡献和潜力分析——聚焦DTP，复制成功模式

**布局**：
- BCG矩阵：城市气泡图（X轴=市场规模/纯销，Y轴=同比增长率）
  - 明星市场（高规模+高增长）
  - 现金牛（高规模+低增长）
  - 问题市场（低规模+高增长）
  - 低潜市场（低规模+低增长）
- 重点城市明细表（长沙/广州等，标注纯销和增长率）
- 核心洞察："聚焦DTP院边店，以[城市A]和[城市B]为复制重点"

#### Slide 10：KA客户健康度诊断（客户维度）
**标题**：核心洞察（二）：需关注下滑的核心大客户，重塑KA合作

**布局**：
- 主图：进销比散点图（X轴=纯销规模，Y轴=SI/SO Ratio，1.0为分界线）
  - 右上象限：核心伙伴/需关注动销
  - 右下象限（SI/SO<1）：健康客户
  - 左上象限（SI/SO>1）：风险/压货客户
- 关键数字：Top2客户贡献了XX%的下滑
- 其他客户增量：XXXX万（正面）

#### Slide 11：F/C产品客户维度拆解
**标题**：产品洞察分解：F和C纯销下滑分析——客户维度

**布局**：两栏
- 左栏（F产品Top客户表）：
  - 列：连锁总部 | 归属省份 | 本期纯销 | 纯销增长额 | 增长率 | 状态
  - 状态用emoji标注：🚀明星 / 🟢稳健 / 🟡持平 / 🔴下滑 / 📉严重失血
- 右栏（C产品Top客户表）：同格式

#### Slide 12：城市维度（F/C产品）
**标题**：产品洞察分解：F和C纯销下滑分析——城市维度

**布局**：
- F产品城市总览表（纯销+增长额+增长率）
- C产品城市总览表
- 洞察：标注"腰斩市场"和"逆势亮点"

#### 洞察汇总页
**标题**：产业务洞察总结

**布局**：三大洞察（编号1/2/3）
1. 核心客户纯销下滑
2. 潜力产品增长良好（R产品DTP机会）
3. 旧产品动销迟缓（Q1/Q2是关键窗口）

### Part 3：战略方向思考（2-3张）

#### 章节标题页
```
大字：战略方向思考
```

#### 战略重构页（核心）
**标题**：战略重构：聚焦资源，打赢三场环环相扣的必赢之战

**三大战役**（必须对应Part 2的三大问题）：

| 战役 | 目标 | 对应洞察 |
|-----|-----|--------|
| 🛡️ KA合作重塑 | 止血 | 核心客户流失 |
| 🚀 R产品DTP深化 | 转型 | 新品渠道机会 |
| ⚡ 前置动销行动 | 增效 | Q1/Q2动销滞后 |

**战略逻辑一句话**：
- 聚焦：重点投入核心KA的F/C动销
- 转型：复制R产品DTP成功模式
- 增效：资源前置到Q1/Q2，提升ROI

#### 行动计划页（可选）
**标题**：下一步行动

| 方向 | 具体方式 |
|-----|--------|
| 头部KA走访 | 走访纯销下滑最严重的X家连锁，明确瓶颈和卡点 |
| DTP合作深化 | 走访当地慢病管理运行良好的头部连锁，了解可复制模式 |

## 视觉设计建议

**色系**：建议使用"深蓝+金黄"或"深蓝+橙红"（医药/商业感）
- 主色：深蓝 `1E3A5F` 或 `003366`
- 强调色：金黄 `F5A623` 或 橙红 `E84B20`
- 背景：白色 `FFFFFF`
- 辅助：浅灰 `F2F4F7`

**数据可视化标准**：
- 下滑数字：红色
- 增长数字：绿色
- 需关注项：橙色
- 趋势图：Sell-in用柱状，Sell-out用折线

**章节标题页**：
- 深色背景（主色），白色大字，左对齐
- 与内容页形成视觉节奏

**表格状态标注**（统一emoji）：
- 🚀 明星（增长>20%）
- 🟢 良好（增长0-20%）  
- 🟡 持平（±5%以内）
- 🔴 下滑（下滑5-30%）
- 📉 严重失血（下滑>30%）
