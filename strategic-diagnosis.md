# 战略诊断模块参考文档

本文档详细说明从数据发现到战略界定的完整方法论。

---

## 🎯 核心方法论

### 三阶段框架

```
Phase 1: 数据洞察
    ↓ (提炼关键发现)
Phase 2: 战略诊断 ← 本文档重点
    ↓ (设计解决方案)
Phase 3: 解决方案
```

### Phase 2 详细流程

```
Step 1: 界定核心战略问题
   ├─ 从数据发现中提炼问题
   ├─ SMART原则检查
   └─ 生成问题陈述表

Step 2: 分解战略议题
   ├─ 逻辑树分解
   ├─ MECE检查
   └─ 议题结构化

Step 3: 优先排序
   ├─ 2×2影响力×可行性评估
   ├─ 优先级排序
   └─ 筛选Top 3-5关键议题

Step 4: 根因诊断
   ├─ 5个为什么
   ├─ 鱼骨图分析
   └─ 根因识别
```

## 📋 Step 1: 界定核心战略问题

### 从数据发现到问题界定

**数据发现示例**（来自7个分析模块）：
1. 全年业绩：Sell-in XX亿（-19%），Sell-out XX亿（-15%）
2. 产品矩阵：F产品-19%，C产品-22%，R产品+53%
3. KA健康度：Top2客户贡献60%下滑
4. 城市维度：F产品在15/20个城市下滑

**问题界定流程**：

#### 1.1 提炼初步问题

从数据发现中提炼3-5个初步问题：
- Q1: 如何阻止KA客户下滑？
- Q2: 如何延缓F/C产品下滑？
- Q3: 如何加速R产品增长？

#### 1.2 SMART原则精炼

将初步问题转化为SMART问题陈述：

**问题1（KA失血）**：
```
问题陈述：
  在Q2末（6月30日前），如何通过主动走访和动销支持，
  使Top2客户（XX连锁和YY连锁）的纯销下滑幅度收窄至5%以内，
  恢复约XXXX万的销售额？

SMART检查：
  ✅ Specific: 明确了Top2客户和下滑幅度目标
  ✅ Measurable: 可量化（下滑幅度<5%，恢复XXXX万）
  ✅ Actionable: 有明确的行动方向（走访+动销支持）
  ✅ Relevant: 直接影响Q2业绩，是短期止血关键
  ✅ Time-framed: Q2末（6月30日前）
```

**问题2（老品防守）**：
```
问题陈述：
  在H1末（6月30日前），如何通过头部市场锁定、失血市场止损、
  成功模式复制三大策略，使F产品下滑趋势延缓至-15%以内，
  C产品在城市层面的分化不再扩大？

SMART检查：
  ✅ Specific: 明确了三大策略和双产品目标
  ✅ Measurable: F产品-15%，C产品分化不扩大
  ✅ Actionable: 有具体策略（锁定+止损+复制）
  ✅ Relevant: F/C是主要收入来源，必须守住
  ✅ Time-framed: H1末（6月30日前）
```

**问题3（新品机会）**：
```
问题陈述：
  在2025年末（12月31日前），如何通过复制长沙等成功模式，
  将R产品在DTP渠道的渗透率从5%提升至10%，
  使其成为中长期增长引擎，贡献XXXX万增量？

SMART检查：
  ✅ Specific: 明确了DTP渠道和渗透率目标
  ✅ Measurable: 渗透率5%→10%，增量XXXX万
  ✅ Actionable: 有成功模式可复制
  ✅ Relevant: 中长期增长引擎，需提前布局
  ✅ Time-framed: 2025年末（12月31日前）
```

#### 1.3 生成问题陈述表

```markdown
## 战略问题清单（Problem Statement）

### 问题1：KA客户失血止血
- **核心问题**：Top2客户贡献了60%的下滑，需在Q2止血
- **现状/背景**：
  - Top2客户纯销下滑XXXX万
  - 进销比>1.3，存在渠道蓄水
  - 其他客户有XXXX万增量
- **决策者**：销售总监 + 区域经理
- **成功标准**：
  - Q2末Top2客户SO降幅收窄至<5%
  - 恢复销售额XXXX万
- **范围**：
  - 包括：Top2客户（XX连锁、YY连锁）
  - 不包括：其他中小客户
- **限制因素**：
  - 时间：Q2仅3个月
  - 资源：销售团队人力有限
  - 竞争：竞品可能同时发力

### 问题2：老品防守策略
- **核心问题**：F/C产品全国性下滑，需H1稳住基本盘
- **现状/背景**：
  - F产品同比-19%，15/20城市下滑
  - C产品同比-22%，10/20城市腰斩
  - 是当前主要收入来源
- **决策者**：市场总监 + 产品经理
- **成功标准**：
  - F产品下滑延缓至-15%以内
  - C产品城市分化不再扩大
- **范围**：
  - 包括：F/C产品，全国市场
  - 不包括：R/N/B产品
- **限制因素**：
  - 市场竞争激烈
  - 资源需与R产品平衡

### 问题3：新品增长加速
- **核心问题**：R产品增长快但基数小，需复制成功模式
- **现状/背景**：
  - R产品同比+53%，但渗透率仅5%
  - 长沙等5城贡献70%增长
  - DTP渠道接受度高
- **决策者**：产品总监 + 渠道总监
- **成功标准**：
  - 年内渗透率提升至10%
  - 贡献增量XXXX万
- **范围**：
  - 包括：R产品，DTP/院边店渠道
  - 不包括：电商渠道
- **限制因素**：
  - 成功模式尚未完全总结
  - 复制需要时间验证
```

## 🌳 Step 2: 分解战略议题

### 逻辑树分解方法

#### 2.1 选择分解框架

根据问题类型选择合适的分解框架：

| 问题类型 | 推荐框架 | 示例 |
|---------|---------|------|
| 客户下滑 | 客户维度分解 | 按客户规模/区域/渠道类型 |
| 产品下滑 | 产品维度分解 | 按产品线/城市/渠道 |
| 增长机会 | 增长驱动分解 | 客户数×单店产出×复购率 |
| 渠道问题 | 渠道维度分解 | DTP/院边店/社区店/电商 |

#### 2.2 构建逻辑树

**问题1（KA失血）的逻辑树**：
```
Top2客户下滑如何止血？
├─ 立即行动（Q2，0-3个月）
│   ├─ 客户深度访谈
│   │   ├─ 明确动销瓶颈
│   │   └─ 了解竞品情况
│   ├─ 动销支持计划
│   │   ├─ 门店促销活动
│   │   ├─ 店员培训计划
│   │   └─ 库存优化建议
│   └─ 高层关系维护
│       ├─ 高层会晤
│       └─ 战略对齐
├─ 中期优化（H2，3-6个月）
│   ├─ 合作模式优化
│   │   ├─ 联合生意计划
│   │   └─ 资源共享机制
│   └─ 精细化运营
│       ├─ 数据共享
│       └─ 联合 forecasting
└─ 长期深化（2025全年）
    ├─ 战略合作升级
    │   ├─ 独家/优先供应
    │   └─ 联合市场推广
    └─ 供应链协同
        ├─ VMI库存管理
        └─ 联合需求预测
```

#### 2.3 MECE检查

**相互独立性检查**：
- ✅ "立即行动" vs "中期优化" vs "长期深化"：时间上不重叠
- ✅ "客户访谈" vs "动销支持" vs "高层关系"：行动类型独立
- ⚠️ "联合生意计划"可能与"战略合作升级"重叠 → 建议合并或明确区分

**完全穷尽性检查**：
- ✅ 已覆盖：时间维度（短期/中期/长期）+ 行动维度（调研/支持/关系/合作）
- ⚠️ 可能遗漏：激励机制（返利/返点）、团队调整（人员配置）

**调整建议**：
- 添加"激励优化"子节点
- 明确"联合生意计划"（中期）vs "战略合作升级"（长期）的边界

## 📊 Step 3: 优先排序

### 2×2优先矩阵

#### 3.1 评估标准

**影响力（Impact）**：
- 🔴 高影响：直接影响营收（>XXXX万）或核心市场
- 🟡 中影响：间接影响或增量较小
- 🟢 低影响：边缘影响

**可行性（Feasibility）**：
- ✅ 高可行性：有明确路径、资源充足、成功案例
- 🟡 中可行性：需要协调资源或跨部门合作
- 🔴 低可行性：资源受限或不确定性高

#### 3.2 矩阵评估

| 议题 | 影响力 | 可行性 | 定位 | 优先级 |
|------|--------|--------|------|--------|
| KA客户失血止血 | 🔴 高（Top2=60%下滑） | ✅ 高（可直接接触） | ⭐ 优先处理 | 1 |
| F产品全国防守 | 🔴 高（主要收入） | 🟡 中（需跨区域协调） | 重点规划 | 2 |
| R产品DTP复制 | 🟡 中（基数小） | ✅ 高（有成功案例） | 快速解决 | 3 |
| C产品城市止损 | 🟡 中（次核心） | 🟡 中（需精准策略） | 暂时搁置 | 4 |

#### 3.3 优先级排序结果

```
🔴 最高优先级（立即启动）：
  1. KA客户失血止血
     → 理由：高影响×高可行性，Q2止血见效快

🟡 次高优先级（本周启动）：
  2. F产品全国防守
     → 理由：高影响×中可行性，需系统化规划
  3. R产品DTP复制
     → 理由：中影响×高可行性，Quick Win

🟢 可暂缓（纳入观察）：
  4. C产品城市止损
     → 理由：中影响×中可行性，待前3项稳定后再处理
```

## 🔍 Step 4: 根因诊断

### 5个为什么（5 Whys）

#### 4.1 问题1的根因分析

**现象**：Top2客户纯销下滑XXXX万

**为什么1**：为什么客户纯销下滑？
→ 客户进货减少（Sell-in -18%）

**为什么2**：为什么客户进货减少？
→ 客户门店动销不畅，库存积压

**为什么3**：为什么门店动销不畅？
→ 门店客流下降15%，竞品冲击加大

**为什么4**：为什么门店客流下降？
→ 社区封控影响 + 竞品在社区店促销力度大

**为什么5**：为什么竞品促销力度大？
→ 竞品今年战略调整，重点投入社区店

**根因识别**：
```
直接原因：客户门店动销不畅
根本原因：竞品战略调整，重点投入社区店
深层原因：我方未及时应对竞品策略变化
```

#### 4.2 鱼骨图分析

```
                     Top2客户下滑
                          │
        ┌─────────────────┼─────────────────┐
        │                 │                 │
     市场因素            客户因素          我方因素
        │                 │                 │
   ┌────┴────┐       ┌────┴────┐       ┌────┴────┐
竞品策略   社区封控  动销不畅  库存积压  促销滞后  人员不足
   │          │        │         │        │         │
竞品加大    疫情影响  客流下降   SI/SO>1.3  未及时   区域人员
社区投入   不可控    -15%               调整      配置不足
```

**根因分类**：
- **可控因素**：促销滞后、人员不足 → 立即行动
- **部分可控**：动销不畅（可通过支持改善）→ 中期优化
- **不可控**：社区封控、竞品策略 → 接受并应对

## 🎯 实战案例

### 案例：某医药零售企业GTM战略诊断

**背景**：
- 2025年Q1业绩复盘
- 整体下滑19%，需要制定Q2-H1策略

**数据发现**：
1. Sell-in XX亿（-19%），Sell-out XX亿（-15%）
2. Top2客户贡献60%下滑
3. F产品-19%，C产品-22%，R产品+53%
4. F产品在15/20城市下滑

**战略问题界定**：
```
问题1（KA失血）：
  如何在Q2末使Top2客户SO降幅收窄至5%以内？

问题2（老品防守）：
  如何在H1末使F产品下滑延缓至-15%以内？

问题3（新品增长）：
  如何在年末使R产品DTP渗透率从5%提升至10%？
```

**优先排序**：
```
1. KA失血止血（高影响×高可行）→ 立即启动
2. F产品防守（高影响×中可行）→ 本周启动
3. R产品复制（中影响×高可行）→ Quick Win
```

**行动计划**：
```
Q2（4-6月）：
  Week 1-2: Top2客户深度访谈
  Week 3-10: 动销支持计划
  Week 11-12: 效果评估

H1（4-6月）：
  同步启动F产品防守策略
  - 头部市场锁定（5个城市）
  - 失血市场止损（3个城市）
  - 成功模式复制（2个城市）

H2（7-12月）：
  R产品DTP复制
  - 长沙模式总结
  - 10个城市复制
  - DTP门店拓展
```

**预期成效**：
```
Q2末：
  - Top2客户止血，恢复XXXX万
  - 整体下滑收窄至-10%

H1末：
  - F产品稳住基本盘
  - R产品渗透率提升至7%

年末：
  - R产品渗透率10%
  - 贡献增量XXXX万
  - 整体恢复增长
```
